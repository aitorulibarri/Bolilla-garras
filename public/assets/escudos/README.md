## Especificaciones Técnicas

- **Formato**: PNG
- **Resolución**: 512x512px (aproximadamente)
- **Peso máximo**: 500KB (todos cumplen)
- **Nombrado**: nombre_equipo.png (minúsculas, sin espacios, guiones bajos)

## Instrucciones para Completar

Para descargar los escudos faltantes, se recomienda:

1. **Wikipedia**: Buscar "[Nombre del equipo] logo PNG Wikimedia Commons"
2. **footylogos.com**: Visitar https://www.footylogos.com/logos/[nombre-equipo]
3. **football-logos.cc**: Visitar https://football-logos.cc/spain/[nombre-equipo]
4. **Sitios oficiales de los clubes**: Muchos equipos tienen su escudo disponible en la sección de prensa o marca
